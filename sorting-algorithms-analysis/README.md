# Sorting Algorithms Analysis

## Description
This project implements and analyzes different sorting algorithms to evaluate their performance on a simulated dataset. The system sorts data based on multiple criteria and compares the efficiency of each algorithm.

## Features
- Implementation of Bubble Sort, Merge Sort, and Quick Sort
- Two-level sorting based on multiple criteria
- Performance analysis using iteration and recursion counters
- Comparison of sorting results between algorithms
- File-based input and output processing

## Algorithms
- Bubble Sort
- Merge Sort
- Quick Sort

## Concepts Used
- Algorithm analysis
- Time complexity comparison
- Multi-criteria sorting
- File I/O

## Technologies
- Python

python hw05.py
