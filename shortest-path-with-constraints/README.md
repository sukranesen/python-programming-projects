# Shortest Path with Constraints

## Description
This project implements a pathfinding algorithm to determine the shortest route across a grid while avoiding obstacles (sinkholes). The system calculates the optimal path based on movement costs and constraints.

## Features
- Grid-based pathfinding system
- Recursive exploration of possible routes
- Cost-based decision mechanism
- Obstacle avoidance (sinkholes)
- Determination of shortest valid path
- File-based input and output processing

## Concepts Used
- Recursion
- Pathfinding algorithms
- Cost optimization
- Grid-based problem solving

## Technologies
- Python

## How to Run
```bash
python route_finder.py input.txt output.txt
```
