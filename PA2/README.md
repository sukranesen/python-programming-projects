# In-Memory Database System

## Description
This project implements a simple in-memory database system that supports SQL-like operations. It processes commands from an input file and performs data manipulation and querying without using an external database.

## Features
- Create and manage tables with dynamic columns
- Insert, delete, update, and query records
- Support filtering using conditions
- Perform JOIN operations between tables
- Count records based on conditions
- Handle invalid operations using exception handling
- Process command-based input sequentially

## Concepts Used
- Data structures (lists, dictionaries)
- Query processing logic
- Exception handling
- File I/O

## Technologies
- Python

## How to Run
```bash
python database.py input.txt
```
